## Usage Tips

### When Library Not Found

If a library isn't in this list:
1. Use `Context7:resolve-library-id` with the library name
2. Check trust score and snippet count
3. Add to your personal reference if used frequently

### Version Selection

- Use latest version (no version suffix) for new projects
- Specify version for:
  - Production stability
  - Legacy project maintenance
  - Version comparison tasks

### Topic Selection

Common effective topics by library type:

**Frameworks**:
- routing, middleware, data fetching, configuration

**UI Libraries**:
- components, theming, customization, accessibility

**State Management**:
- store setup, selectors, middleware, persistence

**Validation**:
- schemas, validation, error handling, transformations

**Testing**:
- setup, assertions, mocking, async testing

### Multi-Library Projects

Common combinations:
- Next.js + Supabase + Tailwind CSS
- React + TanStack Query + Zustand
- Next.js + Prisma + NextAuth
- React + React Hook Form + Zod

For integrations, fetch focused topics:
- Next.js: "middleware authentication"
- Supabase: "client setup browser"
- Tailwind: "responsive design dark mode"
