## Task Workflow

```
backlog/ → active/ → completed/
```

1. **Backlog**: Tasks defined but not started
2. **Active**: Currently in progress (max 2-3 at a time)
3. **Completed**: Done and verified

## Creating Tasks

Use the task template:
```bash
cp tasks/templates/TASK-TEMPLATE.md tasks/backlog/TASK-XXX-slug.md
```

Or use the `/execute-task` command to work on tasks systematically.

## Naming Convention

```
TASK-{number}-{slug}.md
```

Examples:
- `TASK-001-setup-brain-service.md`
- `TASK-002-agency-lead-intake.md`
